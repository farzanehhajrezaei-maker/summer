# Summer — A Paper Summarization Chatbot (Dify)

**Summer** is a lightweight chatbot built in the Dify workspace.  
Its purpose is simple:  
You provide a paper’s **title** and **abstract**, and Summer generates a **one-sentence summary** based on your input.

This repository contains the configuration file (`summer.yml`) used to run the bot within Dify.

## Features
- Generates concise one-sentence summaries of academic papers  
- Works directly from the title + abstract  
- Clean, minimal workflow built using Dify  
- YAML-based logic for easy modification and reuse

## How to Use
1. Log in to your **Dify** workspace  
2. Create a **New App**  
3. Select **Import Workflow**  
4. Upload the `summer.yml` file  
5. The bot will instantly load and be ready for use
